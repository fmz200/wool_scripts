# "reject"        策略返回 HTTP 状态码 404,不附带任何额外内容

# 常见问题：

# 只需要去掉反斜杠：^https?://ad.12306.cn/ad/ser/getAdList
# 不用再说了吧，ad.12306.cn就是想要的主机名
# 2️⃣：有些常用的软件为什么没有去广告配置？
